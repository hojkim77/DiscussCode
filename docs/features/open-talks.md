# Open Talks — "그냥, 개발자끼리 이야기해요"

**MVP: YES**

## Source
유저 직접 작성.

## Post Format
- 제목 + 본문(마크다운, 코드블록 하이라이팅) + 태그(최대 5개)
- 유형: `Discussion` / `Question` / `Show & Tell` / `Meme`
- 선택: 코드블록, 이미지 첨부

## Write Flow
1. 임시 저장 → 미리보기 → 발행
2. 작성 후 24h 이내 수정 가능, 이후 "edited" 표시만

## Question 모드
`#question` 태그 시 답변형 UI 전환 → 작성자가 베스트 답변 지정 가능

## Comments
- 중첩 최대 4뎁스, 그 이하는 "스레드 펼치기"
- 마크다운 + 코드블록 + 이미지/GIF
- 👍/👎 투표 + 이모지 반응
- 정렬: Best / New / Top / Controversial
- `@username` 멘션 → 알림

## List Page Special
- "새 토론 시작" 플로팅 버튼 (다른 카테고리는 시스템 자동 생성이므로 Open Talks 전용)
- 필터: 태그, `#question` only, 미답변 only
